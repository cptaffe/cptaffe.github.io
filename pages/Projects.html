Mesher

STL editor written completely in JS: Mesher. Uses THREE.js and around 1k lines for undo/redo, some simple tools, etc. Haven't worked on it in a while and it is still a bit buggy.

Markdown Editor

Markdown preview page, made awesome with Bootstrap: editor. It dynamically updates the rendered html to match your markdown input, has a couple of save and update key-combos (useful if dynamic-update is turned off becuase document is too long). It also saves to your local memory, keeping the last document the next time you go to the page. Basically a legitimate app with no server-side code whatsoever.
